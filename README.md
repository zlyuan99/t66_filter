# T66 Filter

一个用于优化浏览体验的用户脚本，主要功能包括：

## 功能特点

- 广告过滤：移除各类广告内容
- 图片优化：确保图片正常显示
- 弹窗拦截：阻止恼人的弹窗
- 界面优化：提供更清爽的浏览体验

## 安装方法

1. 首先安装一个用户脚本管理器（如 Tampermonkey）
2. 点击 [安装脚本](t66y_filter_userscript.js)

## 使用说明

安装后脚本会自动运行，无需额外配置。主要功能：

- 自动过滤广告内容
- 优化图片显示
- 移除退出确认对话框
